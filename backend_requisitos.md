# Backend

## Opção A - Python 
    Framework: Flask. 
    Uso de bibliotecas de validação de dados. 

## Opção B - JavaScript 
    Frameworks: Node.js com Express ou Fastify. 
    Uso de bibliotecas de validação de dados. 

## Requisitos comuns de Backend: 
    Banco de Dados: MySQL, PostgreSQL ou SQLite.

    Integração com LLM: implementar a chamada à API escolhida. 

    Criar um prompt claro para garantir que a IA responda no formato correto, preferencialmente JSON. 

    Segurança: a chave da API não deve ficar hardcoded no código, devendo ser configurada por variáveis de ambiente.
